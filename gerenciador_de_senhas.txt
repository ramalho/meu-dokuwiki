====== Gerenciador de senhas ======

Um gerenciador de senhas é um aplicativo que armazena seus logins e senhas em um arquivo encriptado. Isso facilita o uso de senhas fortes e diferentes para cada site, porque você só precisa memorizar uma senha: a senha para abrir o seu gerenciador de senhas.

Para criar uma senha forte para o gerenciador de senhas, eu uso o método [[Dadoware]].

===== Qual gerenciador de senhas devo usar? =====

A Wikipedia em inglês tem uma longa [[https://en.wikipedia.org/wiki/List_of_password_managers|lista de gerenciadores senhas]]. Alguns são software livre, outros são proprietários. Entre os proprietários, alguns têm versões gratuitas.

Eu uso o [[https://1password.com/|1password]], que é um dos gerenciadores recomendados pela [[https://ThoughtWorks.com|ThoughtWorks]]. O 1password funciona super bem no meu computador do trabalho, que é um MacBook Pro. Também existem versões para Windows, iOS e Android. No GNU/Linux, o 1password precisa ser instalado como um plugin para o Google Chrome, que também funciona OK. Para usar o 1password é preciso pagar uma assinatura anual de USD 36. O arquivo de senhas encriptado é sincronizado entre todos os dispositivos.

Também conheço pessoas que usam o [[https://keepassxc.org/|KeepassXC]] (GPLv2) e o [[https://www.lastpass.com/|LastPass]] (proprietário, com versão gratuita).
